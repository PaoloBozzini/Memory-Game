# 🌸 ReMi Dansk 🌸  
### A Cozy Memory Card Game for Learning Danish

> **ReMi Dansk** is a cheerful, web-based memory card game designed to help you learn Danish vocabulary while having fun and sharpening your memory.

🎓 This project was developed as part of the **Hack Your Future Denmark – Foundation Course**.

---

## 🌊 About the Game

Set against a calm Nordic landscape, **ReMi Dansk** combines vocabulary learning with classic memory gameplay.

The objective is simple:

> 🃏 Match all pairs of cards by flipping them over and remembering their positions.

Train your brain. Expand your Danish vocabulary. Enjoy the game.

---

## ✨ Features

- ⏱ **Live Timer**
- 🔄 **Reveals Counter**
- 🗂 **Cards loaded dynamically from SQLite via API**
- 🔁 **Restart Button**
- 🏆 **Game Result Screen with Statistics**
- 📱 **Responsive Design** (Desktop, Tablet & Mobile)

---

## 🎮 How to Play

1. Click **Start Game**  
2. Flip two cards at a time  
3. Find and match all pairs  
4. Try to finish with the fewest reveals and fastest time
